# jekyll-bootstrap4-theme
This is my Bootstrap 4 theme for my Jekyll website.

# Notes
This repository is based off the work in 
<https://simpleit.rocks/how-to-add-bootstrap-4-to-jekyll-the-right-way/>.

# Testing locally
To test the site locally:

1. `$ git clone https://github.com/marcanuy/jekyll-bootstrap4.git`
2. `$ cd jekyll-bootstrap4/` and Install [yarn](https://yarnpkg.com/lang/en/docs/install) components.
3. `yarn install`
4. `jekyll serve`
